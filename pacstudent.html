PACSTUDENT VS. THE MUTANT SLIME HORROR

PACSTUDENT Vs. THE MUTANT SLIME HORROR (we'll just shorten it to Pacstudent) is a spin on the classic arcade game Pac-Man.

One of the most interesting programming subjects I have done is Introduction to Computer Game Development. In that subject, I made a game called "PACSTUDENT VS. THE MUTANT SLIME HORROR", which is a spin on the classic arcade game Pac-Man. This game was created using the Unity game engine, with scripts written in C#. Below are some images of the game.
