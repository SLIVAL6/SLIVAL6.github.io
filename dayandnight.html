Day and Night

Development Period March 2023 - June 2023
Project Links
Relevant Skills
Collaborators

Day and Night is an environment generator with a day and night cycle. It was created for the university subject 31264 Computer Graphics.

The program procedurally generates a small forest environment, and features a time of day system that determines the colour of the sky, the position of the Sun and Moon, and the procedural generation of clouds during the day and stars during the night. It utilises Three.js, a JavaScript library that is used to create 3D computer graphics in WebGL.

For this project, I assumed the role of project lead. I was responsible for the implementations of the following:

  • Day/night cycle system
  • Sun and Moon
  • GUI
  • Lighting
  • Cloud generation
  • Glass encapsulating the environment
  • Source control and branch merging

Almost all features have values and settings that can be changed through the UI for a customised experience. For example, the density, size, opacity, and number of clouds can be changed.

The position of the Sun and Moon is determined by the time of day value which can be seen in the UI. The time of day can be paused and manually set, and the rate at which it progresses can also be changed.
