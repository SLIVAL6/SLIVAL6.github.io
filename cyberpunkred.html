Cyberpunk RED GM Assistant

Development Period October 2023 - Present
Project Links
Relevant Skills
Collaborators

A data management tool for the TTRPG Cyberpunk RED. This is a software that is designed to be used by Game Masters whilst playing the game. Its purpose is to simplify data management for in-game character attributes and to streamline combat encounters by automating tasks that would normally require manual calculations and input.
